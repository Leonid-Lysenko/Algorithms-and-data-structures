## Кодер и декодер Шеннона

Программа реализует алгоритм Шеннона, включая функции кодирования и декодирования.

## Запуск

make

## Тестирование

make test

## Ручное тестирование

    Перейдите в build:

cd build

    Создайте тестовые файлы:

echo "Abracadabraladygaga03022025" > input

    Запустите программу:

./shannon_coder 

    Проверьте результат:

cat dictionary
cat input
cat decoded
